# Contributing

Contributions to improve the scaffold are welcome.

## What's worth contributing

- Improvements to the setup scripts
- New or updated `.claude/` commands and agents
- Documentation fixes and clarifications
- Additional shadcn-svelte components or CSS utilities
- Bug fixes in the GitHub Actions workflows

## How to contribute

1. Fork the repo
2. Create a branch: `git checkout -b my-improvement`
3. Make your changes
4. Commit and push
5. Open a PR against `main`

## Guidelines

- Keep scripts cross-platform where possible (macOS/Linux)
- Keep commands in `.claude/commands/` generic — no project-specific references
- Test setup scripts against a fresh clone before submitting
- Update `thoughts/tech-stack.md` if you change the stack
